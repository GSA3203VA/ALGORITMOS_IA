# Tarefa de Clusterização

Utilizar uma base de dados real, preferencialmente uma que tenha alguma relação com a área de vocês e que vocês detenham conhecimento dos dados;
Apresentar uma breve descrição da base de dados com as devidas etapas de pré-processamento, quando aplicável (eliminação de dados inválidos, normalização, etc);
Definir um objetivo de trabalho e aplicar um ou mais métodos de clusterização vistos na disciplina:  K-Means, K-Medoids, Agrupamento Hierárquico e métodos de densidade (DBSCAN/HDBSCAN).
Avaliar os resultados de forma quantitativa (através do cálculo e apresentação de uma ou mais métricas de avaliação de desempenho) e qualitativa (silhueta, Davies-Bouldin, Calinski-Harabasz, indice Rand) se aplicavel a base de dados;
