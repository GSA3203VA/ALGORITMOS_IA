# Tarefa de ADEC - Analise exploratória e Engenharia Caracteristica

Executar uma EDA (Exploratory Data Analysis) usando metricas para entender a distribuição e qualidade entre as variavíes antes de criar um modelo de IA.
Faça uma EDA de knock com hidrogênio com:
Média, mediana, mínimo, máximo e desvio padrão de cada variável;
Boxplots para lambda, IMEP, avanço, CA50, MAPO, SEPO e KI;
Verificação de dados faltantes, duplicados e pontos fisicamente incoerentes;
Histogramas para observar distribuição e desbalanceamento entre ciclos com e sem knock;
Matriz de correlação Pearson e Spearman;
Gráficos de dispersão, por exemplo:
lambda × SEPO;
avanço × MAPO;
IMEP × KI;
relação de compressão × ocorrência de knock.
