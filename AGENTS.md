# Regras do projeto ALGORITMOS_IA

- Trabalhar preferencialmente com Python.
- Usar a virtualização existente em `.venv`.
- Não alterar arquivos fora da pasta solicitada.
- Não apagar arquivos sem confirmação.
- Não executar `git commit`, `git push` ou `git reset` sem autorização.
- Antes de criar ou alterar códigos, explicar o plano resumidamente.
- Em notebooks Jupyter, comentar o código em português.
- Ao instalar novas bibliotecas, atualizar `requirements.txt`.
- Salvar gráficos e resultados na pasta `RESULTADOS/`, quando ela existir.
